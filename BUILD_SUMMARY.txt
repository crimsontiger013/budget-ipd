SETUP INSTRUCTIONS
================================================================================

QUICKEST START (5 MINUTES):
1. npm create vite@latest my-app -- --template react
2. npm install recharts lucide-react
3. Copy ipd_budget_app.jsx to src/components/
4. Update src/App.jsx to import and use component
5. npm run dev

VERCEL DEPLOYMENT (10 MINUTES):
1. Create Next.js project
2. Copy ipd_budget_app.jsx to app/components/
3. Update app/page.jsx
4. git push to GitHub
5. Connect repo to Vercel
6. Deploy (one-click)

DOCKER DEPLOYMENT (15 MINUTES):
1. Create Dockerfile (provided)
2. docker build -t ipd-budget .
3. docker run -p 3000:3000 ipd-budget
4. Access at http://localhost:3000

See SETUP.md for detailed instructions for each option.
